# NeriSoft Sistema

Sistema desarrollado desde cero con arquitectura modular.

## Stack inicial

- Python
- Flask
- SQLite
- Bootstrap original
- Jinja con macros
- Git en rama main

## Contrato base

- App modular, no monolítica.
- Bootstrap original sin retoques visuales por pantalla.
- Templates con macros Jinja reutilizables.
- SQLite con migraciones controladas.
- Lógica de negocio fuera de templates.
- Tests por módulo.
